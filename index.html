#GeoBingo

UnOfficial Fun For #FOSS4G Bonn 2016

About

My co-worker Calvin has started to make event bingo cards for conferences he attends. It's a way to add a little more fun to a conference and because players are waiting for some obscure word in talks, it means that atendees are paying more attention. So we decided to take this idea on the road so we are bringing this to FOSS4G.

The Rules

Calvinâ€™s bingo is a game where you get a paper sheet with a printed grid filled with words that are related to the conference. The idea is to mark off a word within the grid when it is used in a talk or Q&A at the conference. Once you have marked off all the words in a row or column you have reached the state of BINGO!!!!

Here are some guidelines:

    • The speaker needs to say the word, so you can prompt them to use a word in the Q&A, but you cannot count you or anyone else saying the word in the Q&A.
    • This game is based on the honor system, so please keep it fun by not cheating.
    • You can only win one prize a day
    • Please tweet a picture of your completed bingo sheet to @cwmma or @guidod
    • Please mark you sheet with the date as soon as you get it
    • If you can, please use the following hashtag #GeoBingo #FOSS4G
